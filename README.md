## Installation

In hubot project repo, run:

`npm install hubot-slack-vote --save`

Then add **hubot-slack-vote** to your `external-scripts.json`:

```json
["hubot-slack-vote"]
```

## Sample Interaction

```
user1>> hubot vote :dog: :cat:
```
